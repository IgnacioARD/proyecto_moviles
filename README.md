# proyecto_moviles
Repositorio para el proyecto final de la materia de Desarrollo para Dispositivos Moviles

En esta aplicación móvil se podrán ver las siguientes ventanas:

Ventana de inicio. Aquí se podra crear una nueva receta para nuestra aplicación. Se pedirán 3 campos, estos son:
* Nombre: El nombre del platillo.
* Ingredientes: Una lista de los ingredientes que se deben utilizar (también se puede  agregar el método de cocción aquí si asi se desea).
* Imagen: Una imagen del platillo una vez realizado.

De igual manera podremos hacer 3 acciones desde esta ventana:
* Elegir Imagen: Si este botón se presiona, podremos elegir entre la cámara y la galeria de nuestro dispositivo para agregar una imagen del platillo
* Agregar Receta a lista: Si este botón se presiona, agregaremos la receta a la lista de recetas ya registradas.
* Ver lista de recetas: Si este botón se presiona, cambiaremos a la ventana de recetas de la aplicación, donde podremos ver la lista de todas las recetas registradas en nuestra aplicación
